About endpoints

Harrito is capable of performing commands, that can be sent to him via discord private messages or in a server.

Warning

Please note that if executed inside a server’s channel, commands must be preceded by a !.

It can be present or not if the command is sent through private messages.

Commands

Here is the complete list of Harrito’s commands:

  • help
  • alert
  • echo <message>
  • configure <parameter> <value>
  • get <resource>
  • shell [args]
